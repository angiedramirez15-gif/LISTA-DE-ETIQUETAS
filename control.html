Etiquetas Control
Etiqueta Estructura Descripcion Propiedades Ejemplo
Input type (text) <input type="text"> Permite al usuario ingresar texto libre, como nombres o palabras. placeholder: texto guía dentro del campo.
value: valor por defecto.
maxlength: número máximo de caracteres.
required: campo obligatorio.
Input type(password) <input type="password"> Permite ingresar contraseñas ocultando los caracteres. placeholder: texto guía.
maxlength: límite de caracteres.
required: campo obligatorio.
Input type(email) <input type="email"> Campo para ingresar una dirección de correo electrónico válida. placeholder: texto guía.
required: hace el campo obligatorio.
pattern: define una expresión regular para validar.
Input type(radio) <input type="radio"> Permite seleccionar solo una opción dentro de un grupo. name: agrupa los botones.
value: valor de la opción.
checked: marca por defecto.
