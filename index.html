Etiquetas estructurales
Etiqueta Descripcion Propiedades Ejemplo
<nav> Define una seccion de navegacion que contiene enlaces s otras paginas o partes del sitio web Contiene listas de enlaces (<ul>, <li>, <a>) y puede usar atributos globales como class o id para aplicar estilos.
<header> Define la cabecera o parte superior de una página o sección. Generalmente contiene el título, logotipo, menú de navegación o encabezados principales. No tiene atributos propios,pero puede incluir elementos como(<h1>,<nav>,<img> o <p>). Puede usar los atributos globales class, id y style para aplicar estilos CSS.
<Footer> Define el pie de página de una página web o de una sección. Suele contener información de contacto, derechos de autor o enlaces finales No tiene atributos específicos. Puede incluir texto, enlaces, logotipos o íconos. Se le pueden aplicar los atributos globales class, id y stylepara personalizar su estilo.
